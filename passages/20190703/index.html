cocoapods库与库依赖冲突解决

集成TokenCore、web3swift.pod出现了库与库依赖版本的报错问题：

报错信息如下：

解决的方法如下：

  1. fork TokenCore到自己的github
  2. 修改 s.dependency “BigInt”, “3.1”
  3. 更改pod ‘TokenCore’为 pod ‘TokenCore’, git: ‘https://github.com/Yxiaogui/token-core-ios.git', commit: ‘c87640c72c1cf1662495cf28a90824c6731f790f’
  4. 重新pod install 解决冲突
