2019年终总结

2019年的我的生活中经历的大事件如下：搬家、加班、欠薪。同时也是感谢这个经历让我第一次主动想回家过年，家现在变成"治疗"我自己最好的地方，思考一下为什么会有这样的经历，然后在家这个相对温暖的地方舔舐一下伤口，争取早日治愈自己。害怕的不是成长，害怕的是未曾经历，没有准备。

搬家：

在外工作的人都会经历找房子、搬家，换新工作了可能还需要搬家。以至于有点烦，然后开始考虑期待自己的下一份工作可以更安定、或者开始思考要不买个房子（很明显买不起）。19年的生活是从搬家开始的，本来是打算续租的，可是中介通知我房东要卖房子。19年底又要搬家，原因还是房东要卖房子，真的是太巧合了，巧合的我自己都怀疑我自己了。特别是19年底，过了1月份距离回家就剩两个星期的时间，找房子也是很难的。关于搬家期待明年这个时候的房东不要卖房子，可以安心的续租。

加班：

作为IT从业者加班对我们来说是很平常的事情，更甚者可以接受持续的几个月的加班。但是今年的经历确实有点猛的，加班加到产品经理去董事长那里投诉项目经理，然后经过调停产品经理被开了，留下来一群小可爱接着加班。唉，小可爱们也是敢怒不敢言。

欠薪：

在此之前从未经历过欠薪，但是今年经历了长达4个月的欠薪。由于薪水都是下个月发上一个月，你会发现很快3个月就过去了，很快之前的那点积蓄也快用完了。小伙伴在聊天的时候听到一个让我更震惊的消息她的老公欠薪了8个月，我的天！！！欠薪是2019年上班族的常态吗？难道"物以类聚"，被欠薪的人通过神奇的魔法力量聚集到了一起？你的2019被欠薪了吗？

关于欠薪现在也没什么期待了，唯一期待的是自己明年换一份工作，不喜欢听到重新开始，但是明年确实需要重新开始了。并不希望表达悲壮、伤感的情绪。

成长：

2019年学习的技能有：

  1. React Native构建双端项目，已发布通过苹果审核。
  2. 学习前端知识，目标做的全栈，移动端要转行前端了。
  3. 学习了Node、Docker基础知识。
  4. 学习打卡Algorithm，买了一个域名指向了自己的blog。
